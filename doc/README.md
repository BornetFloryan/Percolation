# Documentation du projet – Percolation / Feu de forêt

Ce dossier contient l’ensemble de la documentation du projet **Percolation – Feu de forêt**.

L’objectif est de décrire à la fois :
- le modèle mathématique utilisé,
- les choix techniques effectués,
- les méthodes d’étude statistique mises en œuvre,
- et la manière de prendre en main et de reprendre le projet.

## Contenu

- `prise_en_main.md`  
  Guide utilisateur : installation, lancement et utilisation de l’application.

- `modele_mathematique.md`  
  Présentation scientifique du modèle de percolation et de la propagation du feu.

- `etude_statistique.md`  
  Étude expérimentale du phénomène de percolation par simulations Monte-Carlo.

- `architecture_technique.md`  
  Organisation du code, choix techniques et structure logicielle du projet.

Ce découpage permet à un développeur ou à un étudiant de comprendre, utiliser et étendre le projet sans connaissance préalable du code.
